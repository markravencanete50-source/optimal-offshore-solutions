Respecting reduced-motion

Framer Motion exposes useReducedMotion(). Gate every transform/parallax on it and fall back to opacity-only (or no motion). Wrap it once in a hook so all components share the rule.

useMotionSafe.ts
import { useReducedMotion } from 'framer-motion';

// Returns a variant-safe distance: 0 when the user prefers less motion.
export function useMotionSafe(distance = 24) {
  const reduced = useReducedMotion();
  return reduced ? 0 : distance;
}
